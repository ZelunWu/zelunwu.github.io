01 - How to Install MITgcm dependencis

MITgcm requires at least one complier: c language compiler. The most common complier in Unix based OS is gcc. Other dependencies are netcdf-c, ifort etc.

Install Dependencies in MacOS

  1. Install homebrew. You can download the clang/gcc source code and compile it by yourself, which may give you more freedom to specify the install rootdir. However, one easier wat I suggest is to use homebrew to install such dependencies but not compile them by yourself.

    Open your terminal, copy this line and execute it:

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

  2. Install gcc. it is easy to install other dependencies if you already installed homebrew. Again, open your terminal:

    brew install gcc

    gcc includes gfortran.

  3. Install netcdf. I don’t suggest you to compile the netcdf source code by yourself, because netcdf library requires other dependencies: zlib, hdf5 etc. Instead, we can use homebrew to install netcdf again, which will install other dependencis for netcdf automatically.

    brew install netcdf

  4. Install openmpi

    To use mpirun, we should install openmpi

    brew install open-mpi

Install Dependencies in Ubuntu, CentOS, and RedHat

While you are looking for the wikipage for ROMs, you may find that they suggest you to install zlib, hdf5, netcdf-c, netcdf-fortran, gcc, gfortran/ifort by compiling the source code. However, from my own experience, it is not easy. Therefore, similarly, I suggest you to install such dependencies with apt/yum.

The name of netcdf library in Ubuntu and CentOS is deference, howevr, we still can use

sudo apt search libnetcdf

to search for the right name of the package, then:

sudo apt install libnetcdfc-dev

similarly, in CentOS & RedHat, we can do:

sudo yum search netcdf
